Ejercico - SQL + Propuesta

El costo de envios por mes para cada Carrier es de:

   Carrier A = $4.400.000,00
   Carrier B = $4.850.000,00
   Carrier c = $1.200.000,00

La conclusión es que El carrier C de menor capacidad, es el que mayor envios puede realizar a menor costo

La propuesta sería reformular el esquema de envios, le daría mayor incidencia al Carrier C donde podría aumentar la
cantidad de envios con el presupuesto mencionado, en base a los costos.

En este caso se usaría para la consultas de diferentes tablas las Queries con Join para obtener los datos
de la diferentes tablas, como si fuese una misma.
